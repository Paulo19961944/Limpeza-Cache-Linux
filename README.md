# LIMPEZA CACHE LINUX
Esse é um script para limpar o cache do seu Linux. Ele identifica o seu Linux e faz a limpeza do cache

## MODO DE USAR
1. **Digite o comando:** `git clone https://github.com/Paulo19961944/Limpeza-Cache-Linux`
2. **Digite o comando:** `cd Limpeza-Cache-Linux`
3. **Digite o comando:** `chmod +x Limpeza.sh`
4. **Digite o comando:** `./Limpeza.sh`
